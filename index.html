Certainly! Here's an example of a simple Python function that greets a person:

def greet(name):
                            return f"Hello, {name}! Welcome to Python."

                        # Using the function
                        result = greet("Alice")
                        print(result)
                        # Output: Hello, Alice! Welcome to Python.

This function takes a name as an input and returns a greeting message. You can call this function with different names to get personalized greetings.
